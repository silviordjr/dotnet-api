1. dotnet new webapi => cria template para webapi;
2. dotnet add package MySqlConnector --version 0.43.0 => possibilita conexão com banco de dados;
3. dotnet add package Swashbuckle.AspNetCore --version 3.0.0 => documentação da api;
4. dotnet add package Dapper --version 1.50.5 => mapeamento do resultado da consulta do banco;
